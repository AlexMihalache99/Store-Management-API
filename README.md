# Store Management API

A simple backend API for managing products and users. Built with Spring Boot, Spring Security and an in-memory H2 database for local development.

---

## Quick Links

- Project: `store-management-api`
- Source: this repository
- Tests: `./mvnw test`
- Coverage report: `target/site/jacoco/index.html`

---

## Tech Stack

- Java 21
- Spring Boot (Web, Data JPA, Security)
- H2 (in-memory database for development)
- Maven (wrapper included)

---

## Getting Started

Prerequisites:

- Java 21 installed
- use the included Maven wrapper `./mvnw`

Run the application locally:

```bash
./mvnw spring-boot:run
```

The app will be available at:

```
http://localhost:8080
```

## API Endpoints

Base URL: `http://localhost:8080`

Health

```http
GET /health
Response: 200 OK
Body: "OK"
```

Users

```
POST   /users        Create user
GET    /users        Get all users
GET    /users/{id}   Get user by ID
PUT    /users/{id}   Update user
DELETE /users/{id}   Delete user
```

Example create user (curl):

```bash
curl -X POST http://localhost:8080/users \
  -H "Content-Type: application/json" \
  -d '{"username":"admin","password":"admin123","role":"ADMIN"}'
```

Products

```
POST   /products       Create product
GET    /products       Get all products
GET    /products/{id}  Get product by ID
PATCH  /products/{id}  Update product
DELETE /products/{id}  Delete product
```

Example create product (curl):

```bash
curl -X POST http://localhost:8080/products \
  -H "Content-Type: application/json" \
  -d '{"name":"iPhone 17","description":"Apple smartphone","price":999.99,"stockQuantity":10}'
```
